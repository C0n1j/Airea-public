# 🌬️ Airea
**Airea** es un proyecto personal diseñado para la visualización y monitoreo de la calidad del aire y tus pulmones. Este proyecto busca ayudar a la gente fumadora a dejar de fumar, llebando una cuenta de los cigarrillos tomados, dando información como: ahorro monetario, mejora salud pulmones...
🚀 **Puedes ver mas info del proyecto en:** airea.info.c0n1j.es
## ✨ Características principales
 * **Proyecto multiplataforma:** El proyecto está disponible tanto para android e ios y versión web
 * **Interfaz Intuitiva:** Diseño limpio centrado en la experiencia de usuario (UX).
 * **Accesibilidad:** Optimizado para ofrecer información clara de un vistazo.
## 🛠️ Tecnologías utilizadas
El proyecto está construido utilizando un stack moderno para garantizar rendimiento y escalabilidad:
| Componente | Tecnología |
|---|---|
| **Codigo** | Flutter/Dark |
| **Base de datos** | Supabase |
| **Hosting web** | Cloudfer |
